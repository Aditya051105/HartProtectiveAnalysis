About the project

HeartSecure X — Cardiac Intelligence
Powered by Machine Learning

A clinical-grade AI screening tool built to make cardiac risk assessment accessible, fast, and interpretable — without replacing the physician.

Our mission

Democratising cardiac awareness

Cardiovascular disease is the world's leading cause of death. Yet millions live unaware of their risk until a cardiac event forces the issue. HeartSecure X was built to change that.

By combining a clinically-informed feature set with a trained Random Forest classifier, we surface risk signals from routine health data that any individual already has access to.

This is not a replacement for medical care — it's a first step toward it.

System design

What makes it work

🌲

Random Forest ensemble

The model uses hundreds of decision trees trained on diverse cardiac patient records, averaging their predictions to produce a robust, low-variance output that resists overfitting.

⚙️

Dedicated preprocessing pipeline

A sklearn ColumnTransformer pipeline handles numerical standardisation and categorical encoding separately — ensuring inference data is transformed identically to training data.

🗂️

Multi-class classification

Unlike binary heart disease models, HeartSecure X outputs one of five distinct diagnoses — providing actionable, condition-specific guidance rather than a simple positive/negative flag.

🔁

Stateless inference API

Built on Flask, the prediction endpoint is stateless and fast. User data is processed in memory and not persisted unless explicitly saved — keeping the privacy footprint minimal.
